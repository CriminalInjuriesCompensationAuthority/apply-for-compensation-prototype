CSC guidance for Private Beta New online form for non-complex sexual assault applications

This guidance is for CSC advisors dealing with customer calls generated by the new application form.

We will update the guidance:

  • when new issues are identified
  • when we improve and refine the design
  • after each new iteration

Private Beta

Private Beta is the phase when we build a working version of our new online application form. It allows us to test the first iteration with real users.

The test group will be small and controlled, so that we can closely monitor the performance of the new online application form and gather user feedback. This will help us understand how it’s working and what improvements we can make.

Through user testing, gathering feedback and data, and by using it to iterate the design, we will have the confidence to extend it to a wider audience and eventually open it up to all users.

We will invite users into Private Beta in one of 2 ways:

  • Our 3rd party support providers (for example Victim Support, Rape Crisis etc) will provide them with the link to our new service
  • CSC advisors will identify suitable users during calls and provide them with the link or complete the application on their behalf.

First iteration

The first iteration will serve users who:

  • have an email address
  • are victims of sexual assault or abuse
  • select to have their application decided on police evidence only (OCJ)
  • are 18 or over
  • are a British citizen or EU national
  • do not have a representative

Key changes

Some of the keys changes from the existing online application form are:

  • the user will no longer access the Justice Portal
  • they will not create an account
  • they will not be able to save and return to the application later
  • the application summary will not be issued automatically for telephone applications
  • a reduction in questions – shortest journey reduced from 41 to 18, longest journey reduced from 64 to 27.

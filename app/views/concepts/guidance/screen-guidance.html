Non-complex sexual assault applications Screen guidance

This guidance contains screenshots of the application service in the order they appear to the user as well as an A to Z list of each screen on the left.

We've identified key ‘pain points’ where we think that, based on research, the user may struggle. Each screen show this guidance text to help you when the applicant calls.

Most screens are self-explanatory and already contain helpful hints for the user. You should read all of the content on the screen to the user, whether or not you are completing it on their behalf.

Telephone applications

You'll complete telephone applications using the same link to the service as a user.

We'll not be able to differentiate between an online application and a telephone one, which means that a copy of the application summary will not be sent automatically to the user.

If the user requests it, you'll need to access the case in Tempus after submitting the application and send a copy with a covering letter (TA10).

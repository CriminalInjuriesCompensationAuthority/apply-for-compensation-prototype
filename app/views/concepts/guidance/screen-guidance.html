Screen guidance

This guidance includes screenshots of our new service. There are two ways to navigate them:

  • view them from the start, in the order they appear to users
  • use the A to Z menu to go to a particular screen

You should read everything on the screen to users. Most screens are self-explanatory, and some already contain hints.

We used research to identify ‘pain points’, questions that may cause users trauma. These screens have extra guidance to help you support them.

Links

All links on screen open in the same window. This is an accessibility requirement.

Users can get back to the previous page using the back button or arrow in the browser.

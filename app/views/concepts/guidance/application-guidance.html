Application guidance Handling process

Telephone applications

You will use the same service link as users to complete telephone applications.

We cannot separate online and telephone applications in this new service. This means that we will not send application summaries to every user.

If a user asks for a summary, get it from Tempus and send it with a covering letter (TA10).

Some service pages have links to other websites that give users more information or help and support. These include:

  • Relevant place guidance www.gov.uk/guidance/criminal-injuries-compensation-a-guide#incident-location
  • Northern Ireland compensation services www.justice-ni.gov.uk/topics/justice-and-law/compensation-services
  • Victims and witness information www.victimandwitnessinformation.org.uk
  • Scottish victims and witness support www.mygov.scot/victim-witness-support/
  • our privacy notice www.gov.uk/guidance/cica-privacy-notice

You can signpost the user to the URLs if necessary, but you do not need to explain the content on external service pages.

Enquiries from users

The new service includes screens where we anticipate that users may need more help. On screen guidance directs them to call us.

You will have to use your judgement as to how to deal with users who are part of the way through the application. Options may include:

  • only answering their specific question
  • helping them through the rest of the application using the screenshot guide
  • completing the entire application on their behalf

Record user feedback

Complete the private beta application tracker to record:

  • details of all private beta calls
  • issues that users encounter with the service
